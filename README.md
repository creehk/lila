Телеграм-бот с игрой "Лила" для групп.

Игра происходит путем бросания кости в тг чате, при каждом ходе показывается информация о позиции, телеграф с картинкой.
Доступна одиночная игра и игра в группе, можно смотреть историю ходов, историю игр, удалить участника из очереди.

*Лила — это древняя индийская трансформационная игра самопознания и психологии, насчитывающая около 2000 лет. Её поле состоит из 72 клеток, отражающих состояния ума, чакры и жизненные препятствия. Игроки формулируют запрос и с помощью кубика проходят путь от старта к осознанию ответа, анализируя свои внутренние установки.

Версия python: 3.10.11
Зависимости:
 - aiogram==2.25.1
